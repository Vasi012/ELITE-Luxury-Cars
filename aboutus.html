About us

The History behind this website.


ELITE is a family run business that was founded in 2022 by Vasilica Catalin who continues to run the business today.
Vasilica Catalin is a student at Code Institute, where he is learning HTML and CSS.
Using the skills learned to date, he created this website.
The success story of Vasilica Catalin is a fascinating one which began in the late 2016 when Vasi started out as a Saturday and Sunday waiter at weeding events.

By the age of 21 Vasi did not know what to do and he was looking for a new challenge, he found the Code Institute decided that now was the right time to do something to change the course of his life!
In 2022, Vasi has spent more than fifty hours trying to build this website and make it the best it can be.

What is this website about?

This website is about cars, as we can see from the logo we are talking about ‘’Luxury Cars’’.
The website is introducing you to a variety of these luxury car brands.
We can find a gallery section by clicking directly on the logo of any car situated on the first page of the website.
Moving forward we have carousel pictures with some of the latest brands availale at ELITE.
Above the footer we can locate the showroom - of course the website is fiction so in the real life we will not find one of this showroom but

Mayfair seems like a good location to choose!

The footer allows us easy access to all the pages related to the website, including some external links for the social media platforms.
In the right hand of the footer, we can see the showroom opening times, behind this time, there are the hours spent building this website.
Moving forward to the contact us page we can observe a simple layout of how you can get in touch with the business directly via email or phone.

Or some options to get to the (fictional) showroom such as flying(!), driving or using the Underground.
Moreover, by clicking on the ‘Book a test drive’ button we will be directed to a different page where we will be required to complete a form to book the best time for us to drive one of these ‘beauties’.
Please enjoy the time spent on my website!
If you have any suggestions you can send them directly to me on my GitHub profile where I will be more then happy to consider them in the future projects that I will complete during this year at the Code Institute.
